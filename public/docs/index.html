Docs

Start with the CLI. Everything runs locally, and reports can be rendered as human-readable text, JSON, or JSONL for streaming workflows.

Quickstart

tabulensis license activate <KEY>

Activate your license first, then run diffs. Use --format json or --format jsonl to integrate with tooling.

Popular commands

  • tabulensis license status
  • tabulensis license deactivate
  • tabulensis diff a.xlsx b.xlsx --format json
  • tabulensis diff a.xlsx b.xlsx --git-diff
  • tabulensis info workbook.xlsx --queries

Diff presets

Choose --fast for speed or --precise for deeper matching.

Exit codes

  • 0 - files are identical
  • 1 - files differ or report is partial
  • 2 - error

Licensing

Licenses allow up to two active devices. Offline access is granted with a short-lived token.

Activation flow

  • Buy or start a trial from Download
  • Run tabulensis license activate <KEY>
  • Use tabulensis license status to confirm
