# UI Snapshots

Baselines live at `desktop/ui_snapshots/<scenario>/baseline.png` with an optional `baseline.json` containing thresholds and metadata.

Generated artifacts during a run:
- `current.png`
- `current.json`
- `current_ready.json`
- `diff.png`
- `diff.json`
- `runs/<tag>.png`
- `runs/<tag>.json`
- `runs/<tag>.ready.json`
- `runs/<tag>.log`

Commit baseline assets to keep visual regression checks stable.
